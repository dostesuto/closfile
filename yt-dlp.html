YouTube 動画ダウンローダー english

これは、コマンドラインからYouTube動画を直接ダウンロードするためのシンプルなPythonベースのツールです。ダウンロードにはyt-dlpライブラリを使用し、動画と音声ファイルのマージにはffmpegを使用します。

特徴

  • YouTubeから動画をダウンロード。
  • ffmpegを使って動画と音声を自動的にマージ。
  • 利用可能な最良の動画および音声品質を選択。
  • カスタマイズ可能な出力ファイル名。
  • シンプルなコマンドラインインターフェース（CUI）。

必要条件

  • Python 3.6 以上
  • yt-dlp（pipでインストール）
  • ffmpeg（動画と音声のマージに必要）

インストール

ステップ 1: Python のインストール

Python 3.6以上がインストールされていることを確認してください。公式サイトからダウンロードできます： https://www.python.org/downloads/

ステップ 2: 依存関係のインストール

リポジトリをローカルにクローンします：

git clone https://github.com/your-username/yt-video-downloader.git
cd yt-video-downloader

次に、pipを使用して必要なPythonライブラリをインストールします：

pip install -r requirements.txt

または、次のコマンドを使用してyt-dlpとffmpegを手動でインストールできます：

pip install yt-dlp

ffmpeg の場合：

  • Windowsの場合、ffmpeg.orgからダウンロードしてインストール。
  • macOSの場合、Homebrewを使用：brew install ffmpeg
  • Linux（Ubuntu/Debian）の場合、次のコマンドを使用：sudo apt install ffmpeg

ffmpegがシステムのPATHに追加されていることを確認してください。

Pythonスクリプトのダウンロード

Pythonスクリプトをダウンロード

ステップ 3: スクリプトの実行

すべてがインストールされたら、ダウンローダースクリプトを実行できます：

python app.py

YouTube動画のURLを入力するように求められます：

YouTubeのURLを入力してください: https://www.youtube.com/watch?v=kagoEGKHZvU

スクリプトが動画をダウンロードし、最適な動画と音声フォーマットを自動的にマージします。

カスタマイズ

app.pyファイルを変更してダウンロードの動作をカスタマイズできます。例えば、ファイルの出力形式を変更したり、異なる品質を選択したりすることができます。

例えば、最良の音声のみをダウンロードするには、次のようにydl_optsを変更します：

ydl_opts = {
    'format': 'bestaudio',
    'outtmpl': '%(title)s.%(ext)s',
}

トラブルシューティング

エラー: ffmpegがインストールされていません

ffmpegがインストールされていて、システムのPATHに含まれていることを確認してください。詳細なインストール手順については、上記のセクションを参照してください。

エラー: ERROR: You have requested merging of multiple formats but ffmpeg is not installed

このエラーが発生した場合、yt-dlpは動画と音声ファイルをマージするためにffmpegを必要とします。ffmpegをインストールすれば、この問題は解決します。
