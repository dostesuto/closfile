💎TMB {TikTok マスボット} english

もっと欲しい場合は星 ⭐ を押してください

サポートが必要な場合や、初心者プログラマーの方は、こちらのガイドを読むことをお勧めします: 「TikTokMassBotting Python スクリプトの使い方」
更新された動作するAPIが必要です。少し待つ必要があります。
さらに動作するAPIが見つかり次第、より早く公開します。

特徴

  * ビューのボット
  * シェアのボット（パッチされていないので、たくさんシェアできます）
  * 使いやすく、高速
  * Linux / Windowsに対応（macOSでは未テスト）
  * プロキシスクレイパー内蔵

インストール方法

  * pip install requests pystyle==1.5

使い方：

  * 動画チュートリアル : https://www.youtube.com/watch?v=tnHadVigdQQ

  * python <ShareBot.py または ViewBot.py>
  * 動画のURLを貼り付け
  * 数量を選択
  * プロキシをスクレイプしたい場合は「はい」か「いいえ」を入力
  * Proxies.txt に記載されたプロキシタイプを入力
  * プロキシのタイムアウトを入力 | よいタイムアウトを得るには3を選択
  * スレッド数を入力 | スレッド数が多いほど速くなりますが、コンピュータが遅くなります | 100~1000を選択

コツ

  * VPSで使用（速くなる！）
  * HTTPプロキシタイプ、socks4プロキシタイプ、socks5プロキシタイプをそれぞれ3回実行する
